# 💰 Meeting Cost Tracker

A modern, high-performance web application designed to visualize the true cost of meetings in real-time. Stop wasting time and see the financial impact of every second.

## ✨ Features

- **Real-time Cost Counter**: Watch the meeting cost increase every second based on your team's average salary.
- **Dynamic Controls**: Adjust the number of attendees and average yearly salary on the fly.
- **Subtle Elapsed Time**: A discreet ticking clock keeps track of exactly how long the meeting has lasted.
- **Premium Design**: Sleek, glassmorphic UI with smooth, animated gradient backgrounds for a modern look and feel.
- **Localization**: Supports multiple currencies (NOK, GBP, EUR, USD) with proper local formatting.
- **Live Title Updates**: The browser tab title updates automatically with the current cost.

## 🚀 Experience the Visuals

Built with a focus on aesthetics, the app features:
- **Glassmorphism**: Translucent panels with backdrop blurs.
- **Micro-animations**: Subtle pulsing effects when the timer is active.
- **Responsive Layout**: Designed to use the full screen for maximum impact.

## 🛠️ Tech Stack

- **React** for reactive state management.
- **TypeScript** for robust development.
- **Vite** for ultra-fast builds and HMR.
- **Lucide React** for beautiful, minimalist iconography.
